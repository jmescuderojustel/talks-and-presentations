La pareja perfecta: Flexbox y Grid Layout

Anxo Fole y David Gomez 11:30 - 12:15

A través de ejemplos prácticos veremos los elementos clave de dos módulos de CSS muy potentes que se complementan a la perfección para realizar maquetaciones responsive de manera sencilla.

Flexbox adecuado para componentes, que podemos usar en la actualidad y Grid Layout, adecuado para layouts principales de aplicaciones, que pronto podremos usar en la nueva especificación de CSS4.
