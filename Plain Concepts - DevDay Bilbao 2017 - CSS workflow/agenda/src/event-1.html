WebApi, más allá del 'Hola mundo'

Hugo Biarge 9:15 - 10:15

En esta sesión abordaremos algunos de los aspectos que consideramos importantes a la hora de implementar un API Http con WebApi (Full Framework).

Hablaremos de cómo organizar nuestros proyectos y soluciones, el uso de contenedores de inyección de dependencias, la seguridad, el versionado, la documentación, la separación de lecturas y escrituras o el testing como elementos clave a la hora de desarrollar un código que nos facilite el desarrollo, mantenimiento y extensibilidad de nuestras aplicaciones.
