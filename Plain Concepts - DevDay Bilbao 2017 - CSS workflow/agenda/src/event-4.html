Mejorando el workflow de maquetación

Borja Andrés y Jesús María Escudero 12:15 - 13:00

En esta sesión mostraremos algunas sugerencias y trucos que nos ayudaran a mejorar nuestro flujo de trabajo maquetando para ser un poco más productivos.

Enseñaremos varias alternativas para organizar nuestro código CSS, así como buenas prácticas, que nos ayudaran a mantener a lo largo del tiempo nuestra solución de estilos.

Hablaremos también de ciertas herramientas como SASS, LESS o PostCSS que permitirán aumentar nuestra productividad, además de ciertas recomendaciones para que toda nuestra base de código sea compartida por el equipo de manera coherente.
